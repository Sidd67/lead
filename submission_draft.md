# LeadHunter AI: Submission Draft

## Project Title
**LeadHunter AI** — *Build an autonomous web agent using the TinyFish API*

## Theme Selection
**AI Agents for Sales & Recruitment Automation** (or Productivity)

## Idea for Submission
LeadHunter AI is an autonomous lead generation and job sourcing platform that automates the tedious process of finding and validating professional opportunities. Powered by the **TinyFish API**, it deploys an intelligent agent that navigates the web like a human to find high-value job leads and candidate data.

### 🚀 Key Innovations
1. **Autonomous Direct-to-Source Discovery**: Unlike typical scrapers, LeadHunter prioritizes finding official "Direct Career" links, ensuring application integrity.
2. **Invisible Maintenance**: Features a fully automated 4:00 AM job refresh and 30-day cleanup cycle, keeping the pipeline fresh without manual intervention.
3. **Team Context Integration**: Captures rich team descriptions to provide candidate-facing value, helping high-quality talent make informed decisions about joining.
4. **Resilient Architecture**: Built with a decoupled Vite/TypeScript frontend and a robust Drizzle/SQLite backend for high performance and reliability.

### 🛠️ Tech Stack
- **AI Agent**: TinyFish API
- **Frontend**: React, Tailwind CSS, Framer Motion
- **Backend**: Express, Drizzle ORM, SQLite
- **Automation**: Cron-based scheduled maintenance
